# 1.3 Matriz de Trazabilidad IREB

Matriz de trazabilidad que vincula requisitos con sus implementaciones, diseños y pruebas.

## Contenido

- Identificación de requisitos (ID, descripción)
- Trazabilidad hacia:
  - Requisitos relacionados
  - Componentes de diseño
  - Casos de prueba
  - Historias de usuario
- Estado de cobertura
- Responsables

## Propósito

Garantizar que todos los requisitos sean rastreables y estén cubiertos en todas las fases del desarrollo.
