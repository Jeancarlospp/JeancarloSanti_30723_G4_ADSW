# 1. ELICITACION

Fase de elicitación y análisis de requisitos del proyecto. Contiene toda la información recopilada, documentada y validada con los stakeholders.

## Estructura

- **1.0 Linea Base** - Línea base de requisitos establecida
- **1.1 Especificación RS** - Especificación de Requisitos de Software
- **1.2 Cronograma** - Cronograma del proyecto
- **1.3 Matriz IREB** - Matriz de Trazabilidad IREB
- **1.4 Historias de Usuario** - Historias de usuario del proyecto
- **1.5 Actas de Reunión** - Actas de reuniones realizadas
- **1.6 Backlog** - Product Backlog y Sprint Backlog
- **1.7 Pruebas Unitarias** - Casos de prueba unitarias
- **1.8 Reporte de Errores** - Registro de defectos encontrados

## Objetivo

Documentar completamente los requisitos funcionales y no funcionales del sistema, asegurando la trazabilidad desde la elicitación hasta la implementación.
