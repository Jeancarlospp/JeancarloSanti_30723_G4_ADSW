# 1.7 Pruebas Unitarias

Casos de prueba unitarias para validar el funcionamiento de componentes individuales.

## Contenido

- Especificación de casos de prueba
- Datos de entrada y salida esperada
- Condiciones de ejecución
- Resultados y evidencia
- Defectos encontrados
- Trazabilidad a requisitos

## Cobertura

Se busca alcanzar una cobertura mínima del 80% del código implementado con pruebas unitarias.
