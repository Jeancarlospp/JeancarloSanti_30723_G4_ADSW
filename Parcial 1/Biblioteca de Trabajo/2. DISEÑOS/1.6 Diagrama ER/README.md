# 1.6 Diagrama Entidad-Relación (ER)

Modelado de la estructura de la base de datos.

## Contenido

- Entidades del sistema
- Atributos de cada entidad
- Claves primarias
- Claves foráneas
- Relaciones entre entidades
- Cardinalidad (1:1, 1:N, M:N)
- Restricciones de integridad
- Normalizacion (hasta 3FN mínimo)

## Nivel de Detalle

Se presentan en diferentes niveles:
- **MER** (Modelo Entidad-Relación) - Conceptual
- **DER** (Diagrama Entidad-Relación) - Lógico
- **Esquema Relacional** - Físico

## Herramientas

Puede representarse usando:
- MySQL Workbench
- DBDesigner
- Lucidchart
- Draw.io

## Propósito

Proporcionar una guía clara para el diseño e implementación de la base de datos del sistema.
