# 2. PERFIL PROYECTO

Información general y contexto del proyecto.

## Contenido

- Descripción general del proyecto
- Objetivos del proyecto
- Alcance y limitaciones
- Stakeholders principales
- Información del cliente
- Riesgos identificados
- Supuestos
- Restricciones del negocio y técnicas

## Propósito

Proporcionar una visión general del proyecto para que todos los miembros del equipo comprendan el contexto, los objetivos y los límites del trabajo a realizar.
