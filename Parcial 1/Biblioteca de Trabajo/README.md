# Biblioteca de Trabajo

Directorio central que contiene toda la documentación, diseños y artefactos del proyecto.

## Estructura

- **1. ELICITACION/** - Fase de elicitación de requisitos del proyecto
- **2. PERFIL PROYECTO/** - Información del perfil y contexto del proyecto
- **3. DISEÑOS/** - Artefactos de diseño y arquitectura

## Descripción

Esta biblioteca organiza todos los artefactos generados durante las fases de análisis y diseño del software, facilitando la trazabilidad y gestión de requisitos.
