# 1.5 Diagrama de Componentes

Representa la estructura física de la aplicación en términos de componentes y sus dependencias.

## Contenido

- Componentes del sistema
- Interfaces provistas y requeridas
- Dependencias entre componentes
- Puertos de comunicación
- Artefactos asociados
- Ubicación física

## Componentes Típicos

- Componentes de interfaz (UI)
- Componentes de negocio
- Componentes de datos
- Servicios externos
- Librerías y frameworks

## Utilidad

Facilita la comprensión de cómo se integran los componentes del sistema y sus puntos de contacto.
