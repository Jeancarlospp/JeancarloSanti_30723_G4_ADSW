# 1.4 Diagrama de Clases

Modelado de la estructura de clases y sus relaciones.

## Contenido

- Definición de clases
- Atributos de cada clase
- Métodos y operaciones
- Relaciones (herencia, asociación, agregación, composición)
- Multiplicidad
- Visibilidad (público, privado, protegido)
- Interfaces implementadas
- Estereotipos

## Herramientas

Puede representarse usando:
- UML (Unified Modeling Language)
- Diagramas en herramientas como StarUML, Lucidchart, etc.
- Código fuente bien documentado

## Propósito

Proporcionar una visión clara de la estructura orientada a objetos del sistema y sus relaciones.
