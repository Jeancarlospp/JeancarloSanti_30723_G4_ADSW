# 3. DISEÑOS

Artefactos de diseño que especifican la arquitectura y estructura del sistema.

## Estructura

- **1.1 Patron de Diseño** - Patrones de diseño utilizados
- **1.2 Diseño de Arquitectura** - Arquitectura general del sistema
- **1.3 Casos de Uso Extendido** - Descripción detallada de casos de uso
- **1.4 Diagrama de Clases** - Estructura de clases del sistema
- **1.5 Diagrama de Componentes** - Componentes y sus interacciones
- **1.6 Diagrama ER** - Modelo de datos de la base de datos

## Objetivo

Documentar el diseño arquitectónico y de detalle del sistema para facilitar la implementación consistente y el mantenimiento futuro.
