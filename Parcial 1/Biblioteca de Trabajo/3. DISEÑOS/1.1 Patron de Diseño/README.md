# 1.1 Patrón de Diseño

Documentación de los patrones de diseño aplicados en la solución.

## Contenido

- Nombre del patrón
- Descripción y propósito
- Problema que resuelve
- Solución propuesta
- Diagrama del patrón
- Ventajas y desventajas
- Ejemplos de implementación
- Componentes involucrados

## Patrones Comunes

- MVC (Modelo-Vista-Controlador)
- Repository Pattern
- Singleton
- Factory
- Observer
- Strategy
- Decorator
- Adapter

## Beneficio

Permite reutilizar soluciones probadas, mejora la comunicación entre desarrolladores y facilita el mantenimiento.
