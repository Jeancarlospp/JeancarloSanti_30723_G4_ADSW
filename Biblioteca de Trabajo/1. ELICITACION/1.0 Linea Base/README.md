# 1.0 Línea Base

Contiene la línea base de requisitos establecida del proyecto.

## Contenido

- Requisitos funcionales base del sistema
- Requisitos no funcionales iniciales
- Asunciones y restricciones del proyecto
- Aprobaciones y firmas de stakeholders

## Versión

Se mantiene el control de versiones de la línea base y sus cambios de modo que se pueda rastrear la evolución de los requisitos.
